About this space

The journey is the design.

Dravidian architecture was built on one idea — that a space should do more than shelter. Every threshold, every corridor, every carved detail was designed to create a specific experience as you moved through it. The journey was the design.

That thinking shaped how I see everything. This portfolio follows the same logic — not a website, but a world. Each project is a space within it. Each space has a story. And the story is always the point.
